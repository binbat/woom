A painless self-hosted Meeting service, use [Live777](https://github.com/binbat/live777) engine

## Run

Depends: Redis, Live777 service

```bash
docker compose up
```

WebApp:

```bash
npm install
npm run dev
```

Server:

```bash
go run .
```


## Release build

```bash
make
```
